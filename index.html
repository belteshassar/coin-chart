Mass, value and size of coins of common currencies

This graph shows includes coins of the 20 most traded currencies. When more than one version of the same denomination is circulated, I have chosen to include the most recently minted. Hover the mouse over a coin to bring it to the top. Leave the mouse there for a second to see a tooltip with the nominal value and the ISO code of the currency. You can also click a coin to bring it to the back if it obstructs other coins.

Created by Daniel Eriksson using D3.js. Sources for coin data: Respective national mints, Wikipedia, Numista. Exchange rates are from and are provided by the European Central Bank. The code is available on Github.
